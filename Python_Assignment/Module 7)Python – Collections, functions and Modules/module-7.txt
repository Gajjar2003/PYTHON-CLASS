1.Accessing List

    1.Understanding how to create and access elements in a list.
        âœ… Creating a List

        A list in Python is created using square brackets [].
        It can hold multiple values (even of different types).

        # Example of creating lists
        numbers = [10, 20, 30, 40, 50]       # list of integers
        names = ["Alice", "Bob", "Charlie"]  # list of strings
        mixed = [1, "hello", 3.5, True]      # list with mixed data types

        âœ… Accessing Elements

        Lists are ordered, and you can access elements using indexing.
        ğŸ‘‰ Indexing starts at 0 in Python.

        numbers = [10, 20, 30, 40, 50]

        print(numbers[0])   # First element â†’ 10
        print(numbers[2])   # Third element â†’ 30
        print(numbers[-1])  # Last element â†’ 50 (negative indexing)
        print(numbers[-2])  # Second last element â†’ 40

        âœ… Accessing Multiple Elements (Slicing)

        You can use slicing to get a part of the list:

        numbers = [10, 20, 30, 40, 50]

        print(numbers[1:4])   # [20, 30, 40] â†’ from index 1 to 3
        print(numbers[:3])    # [10, 20, 30] â†’ from start to index 2
        print(numbers[2:])    # [30, 40, 50] â†’ from index 2 to end
        print(numbers[::2])   # [10, 30, 50] â†’ every 2nd element

        âœ… Changing Elements

        Lists are mutable, so you can change elements:

        numbers = [10, 20, 30, 40, 50]

        numbers[1] = 99        # change 2nd element
        print(numbers)         # [10, 99, 30, 40, 50]


2. Indexing in lists (positive and negative indexing).

        âœ… Indexing in Lists

        When you create a list, each element gets a position (index number).

        ğŸ‘‰ Positive Indexing â†’ Starts from 0 (left â†’ right).
        ğŸ‘‰ Negative Indexing â†’ Starts from -1 (right â†’ left).

        ğŸ”¹ Example List
        fruits = ["apple", "banana", "cherry", "date", "elderberry"]

        ğŸ”¹ Positive Indexing
        Index:   0        1         2        3          4
        Value: "apple"  "banana"  "cherry" "date"   "elderberry"

        print(fruits[0])   # apple  (first element)
        print(fruits[2])   # cherry (third element)
        print(fruits[4])   # elderberry (fifth element)

        ğŸ”¹ Negative Indexing
        Index:  -5      -4       -3       -2        -1
        Value: "apple" "banana" "cherry" "date"  "elderberry"

        print(fruits[-1])   # elderberry 
        print(fruits[-2])   # date      
        print(fruits[-5])   # apple      

3.Slicing a list: accessing a range of elements.

        âœ… What is List Slicing?

        Slicing means accessing a range of elements from a list using the format:

        list[start:end:step]


        start â†’ index where the slice begins (inclusive).

        end â†’ index where the slice stops (exclusive).

        step â†’ interval (default is 1).

        ğŸ”¹ Example List
        numbers = [10, 20, 30, 40, 50, 60, 70]

        ğŸ”¹ Basic Slicing
        print(numbers[1:4])   # [20, 30, 40] â†’ from index 1 to 3
        print(numbers[:3])    # [10, 20, 30] â†’ from start to index 2
        print(numbers[3:])    # [40, 50, 60, 70] â†’ from index 3 to end

        ğŸ”¹ Using Step
        print(numbers[::2])   # [10, 30, 50, 70] â†’ every 2nd element
        print(numbers[1::2])  # [20, 40, 60] â†’ every 2nd element starting from index 1

        ğŸ”¹ Negative Index with Slicing
        print(numbers[-4:-1])   # [40, 50, 60] â†’ from index -4 to -2
        print(numbers[::-1])    # [70, 60, 50, 40, 30, 20, 10] â†’ reverse the list

2.  List Operations

    1.Common list operations: concatenation, repetition, membership

            âœ… Common List Operations in Python
        ğŸ”¹ 1. Concatenation (+)

        You can join two or more lists using the + operator.

        list1 = [1, 2, 3]
        list2 = [4, 5, 6]

        result = list1 + list2
        print(result)   # [1, 2, 3, 4, 5, 6]

        ğŸ”¹ 2. Repetition (*)

        You can repeat elements of a list using the * operator.

        list1 = ["A", "B"]

        result = list1 * 3
        print(result)   # ['A', 'B', 'A', 'B', 'A', 'B']

        ğŸ”¹ 3. Membership (in, not in)

        You can check whether an element exists in a list.

        fruits = ["apple", "banana", "cherry"]

        print("apple" in fruits)     # True
        print("mango" in fruits)     # False
        print("mango" not in fruits) # True

    2.Understanding list methods like append(), insert(), remove(), pop().


            âœ… Common List Methods
        ğŸ”¹ 1. append() â†’ Add element at the end
        fruits = ["apple", "banana"]
        fruits.append("cherry")
        print(fruits)   # ['apple', 'banana', 'cherry']

        ğŸ”¹ 2. insert() â†’ Add element at a specific position
        fruits = ["apple", "banana"]
        fruits.insert(1, "cherry")   # insert at index 1
        print(fruits)   # ['apple', 'cherry', 'banana']

        ğŸ”¹ 3. remove() â†’ Remove the first occurrence of an element
        fruits = ["apple", "banana", "cherry", "banana"]
        fruits.remove("banana")   # removes first 'banana'
        print(fruits)   # ['apple', 'cherry', 'banana']


        âš ï¸ If the element does not exist â†’ Error (ValueError).

        ğŸ”¹ 4. pop() â†’ Remove element at a specific index (default: last element)
        fruits = ["apple", "banana", "cherry"]

        fruits.pop()       # removes last element â†’ 'cherry'
        print(fruits)      # ['apple', 'banana']

        fruits.pop(0)      # removes element at index 0 â†’ 'apple'
        print(fruits)      # ['banana']

3. Working with Lists

    1.Iterating over a list using loops

            âœ… Iterating Over a List Using Loops
        ğŸ”¹ 1. Using a for loop (most common)
        fruits = ["apple", "banana", "cherry"]

        for item in fruits:
            print(item)


        ğŸ‘‰ Output:

        apple
        banana
        cherry

        ğŸ”¹ 2. Using for loop with range() and indexing
        fruits = ["apple", "banana", "cherry"]

        for i in range(len(fruits)):   # i = 0,1,2
            print(i, fruits[i])        # prints index and value


        ğŸ‘‰ Output:

        0 apple
        1 banana
        2 cherry

        ğŸ”¹ 3. Using a while loop
        fruits = ["apple", "banana", "cherry"]

        i = 0
        while i < len(fruits):
            print(fruits[i])
            i += 1

        ğŸ”¹ 4. Using enumerate() (index + value together, cleaner way)
        fruits = ["apple", "banana", "cherry"]

        for index, value in enumerate(fruits):
            print(index, value)

    2.Sorting and reversing a list using sort(), sorted(), and reverse().

            âœ… Sorting and Reversing Lists
        ğŸ”¹ 1. sort() â†’ Sorts the list in-place

        Changes the original list

        Default: ascending order

        numbers = [40, 10, 30, 20]
        numbers.sort()
        print(numbers)   # [10, 20, 30, 40]


        ğŸ‘‰ Sorting in descending order:

        numbers.sort(reverse=True)
        print(numbers)   # [40, 30, 20, 10]

        ğŸ”¹ 2. sorted() â†’ Returns a new sorted list

        Original list remains unchanged

        Can be used on any iterable (list, tuple, string, etc.)

        numbers = [40, 10, 30, 20]

        print(sorted(numbers))        # [10, 20, 30, 40]
        print(numbers)                # [40, 10, 30, 20] (unchanged)

        print(sorted(numbers, reverse=True))  # [40, 30, 20, 10]

        ğŸ”¹ 3. reverse() â†’ Reverses the list in-place

        Does not sort, only flips the order

        numbers = [40, 10, 30, 20]
        numbers.reverse()
        print(numbers)   # [20, 30, 10, 40]

    3.Basic list manipulations: addition, deletion, updating, and slicing.

                âœ… Basic List Manipulations in Python
            ğŸ”¹ 1. Addition (Adding Elements)

            You can add elements in multiple ways:

            fruits = ["apple", "banana"]

            fruits.append("cherry")           # add at end
            print(fruits)   # ['apple', 'banana', 'cherry']

            fruits.insert(1, "mango")         # add at index 1
            print(fruits)   # ['apple', 'mango', 'banana', 'cherry']

            fruits.extend(["grape", "orange"])  # add multiple
            print(fruits)   # ['apple', 'mango', 'banana', 'cherry', 'grape', 'orange']

            ğŸ”¹ 2. Deletion (Removing Elements)

            Several ways to remove items:

            fruits = ["apple", "banana", "cherry", "banana"]

            fruits.remove("banana")   # removes first 'banana'
            print(fruits)   # ['apple', 'cherry', 'banana']

            fruits.pop()              # removes last element
            print(fruits)   # ['apple', 'cherry']

            fruits.pop(0)             # removes element at index 0
            print(fruits)   # ['cherry']

            del fruits[0]             # delete by index
            print(fruits)   # []

            fruits = ["apple", "banana", "cherry"]
            fruits.clear()            # removes all elements
            print(fruits)   # []

            ğŸ”¹ 3. Updating (Changing Elements)

            Lists are mutable â†’ you can change elements directly.

            numbers = [10, 20, 30, 40]

            numbers[1] = 99          # change second element
            print(numbers)   # [10, 99, 30, 40]

            numbers[2:4] = [77, 88]  # update multiple using slicing
            print(numbers)   # [10, 99, 77, 88]

            ğŸ”¹ 4. Slicing (Accessing a Range of Elements)
            numbers = [10, 20, 30, 40, 50, 60]

            print(numbers[1:4])    # [20, 30, 40]
            print(numbers[:3])     # [10, 20, 30]
            print(numbers[3:])     # [40, 50, 60]
            print(numbers[::2])    # [10, 30, 50]
            print(numbers[::-1])   # [60, 50, 40, 30, 20, 10] (reversed)

4. Tuple

    1.Introduction to tuples, immutability.

            What is a Tuple?

            A tuple is a collection of items, like a list, but enclosed in parentheses () instead of square brackets [].

            # Example of tuples
            numbers = (10, 20, 30, 40)
            fruits = ("apple", "banana", "cherry")
            mixed  = (1, "hello", 3.5, True)

            ğŸ”¹ Key Features of Tuples

            Ordered â†’ Items have a fixed order.

            Allow duplicates â†’ Same value can appear multiple times.

            Heterogeneous â†’ Can store different data types.

            Immutable â†’ Once created, you cannot change, add, or remove elements.

            ğŸ”¹ Tuple vs List
            Feature	List ([])	Tuple (())
            Mutable	âœ… Yes (can change, add, delete)	âŒ No (cannot change after creation)
            Performance	Slower	Faster
            Use case	When data needs updates	When data should stay constant
            ğŸ”¹ Immutability Example
            fruits = ("apple", "banana", "cherry")

            # Trying to update
            # fruits[1] = "mango"   âŒ This gives an error (TypeError)


            ğŸ‘‰ Once created, tuple elements cannot be modified.

            ğŸ”¹ But you canâ€¦

            Access elements using indexing and slicing:

            print(fruits[0])    # apple
            print(fruits[-1])   # cherry
            print(fruits[0:2])  # ('apple', 'banana')


            Reassign the whole tuple (create a new one):

            fruits = ("mango", "orange")
            print(fruits)   # ('mango', 'orange')

    2.Creating and accessing elements in a tuple

        âœ… Creating Tuples
        ğŸ”¹ 1. Empty Tuple

        ğŸ”¹ 3. Multiple Elements Tuple

        ğŸ”¹ 4. Tuple Without Parentheses (Packing)
        t3 = 1, 2, 3, 4

        ğŸ”¹ 5. Mixed Data Types
        t4 = (1, "hello", 3.5, True)

        âœ… Accessing Elements in Tuples

        fruits = ("apple", "banana", "cherry", "date")

        ğŸ”¹ Positive Indexing
        print(fruits[0])   # apple
        print(fruits[2])   # cherry

        ğŸ”¹ Negative Indexing

        print(fruits[:2])    # ('apple', 'banana')
        print(fruits[::2])   # ('apple', 'cherry')
    
    3.Basic operations with tuples: concatenation, repetition, membership.

5. Accessing Tuples

    1.Accessing tuple elements using positive and negative indexing

        âœ… Accessing Tuple Elements with Indexing

        ğŸ‘‰ Tuples are ordered collections, so each element has a fixed index number.

        ğŸ”¹ Example Tuple
        fruits = ("apple", "banana", "cherry", "date", "elderberry")

        ğŸ”¹ Positive Indexing (from left â†’ right)

        Index starts from 0 for the first element.

        Index:   0        1         2        3            4
        Value: "apple"  "banana"  "cherry" "date"   "elderberry"

        print(fruits[0])   # apple   (first element)
        print(fruits[2])   # cherry  (third element)
        print(fruits[4])   # elderberry (fifth element)

        ğŸ”¹ Negative Indexing (from right â†’ left)

        Index starts from -1 for the last element.

        Index:  -5       -4        -3       -2           -1
        Value: "apple" "banana" "cherry" "date"   "elderberry"

        print(fruits[-1])   # elderberry (last element)
        print(fruits[-2])   # date       (second last element)

    2.Slicing a tuple to access ranges of elements.

        âœ… Slicing Tuples

        ğŸ‘‰ Just like lists, we use the syntax:

        tuple[start:end:step]


        start â†’ index where the slice begins (inclusive).

        end â†’ index where the slice stops (exclusive).

        step â†’ interval (default = 1).

        ğŸ”¹ Example Tuple
        numbers = (10, 20, 30, 40, 50, 60, 70)

        ğŸ”¹ Basic Slicing
        print(numbers[1:4])   # (20, 30, 40) â†’ from index 1 to 3
        print(numbers[:3])    # (10, 20, 30) â†’ from start to index 2
        print(numbers[3:])    # (40, 50, 60, 70) â†’ from index 3 to end

        ğŸ”¹ Using Step
        print(numbers[::2])   # (10, 30, 50, 70) â†’ every 2nd element
        print(numbers[1::2])  # (20, 40, 60) â†’ every 2nd element starting at index 1

        ğŸ”¹ Negative Index with Slicing
        print(numbers[-4:-1])   # (40, 50, 60) â†’ from index -4 to -2
        print(numbers[::-1])    # (70, 60, 50, 40, 30, 20, 10) â†’ reversed tuple

6. Dictionaries

    1.Introduction to dictionaries: key-value pairs

            âœ… Introduction to Dictionaries in Python
            ğŸ”¹ What is a Dictionary?

            A dictionary is a collection in Python that stores data in the form of keyâ€“value pairs.

            Keys â†’ unique identifiers

            Values â†’ data associated with the keys

            Dictionaries are enclosed in curly braces {}.

            student = {
                "name": "Alice",
                "age": 20,
                "grade": "A"
            }

            ğŸ”¹ Features of Dictionaries

            Unordered (before Python 3.7 â†’ unordered, now ordered by insertion).

            Keyâ€“Value pairs.

            Keys must be unique (cannot repeat).

            Values can be duplicated.

            Keys are immutable types (like string, number, tuple).

            ğŸ”¹ Example
            person = {
                "name": "John",
                "age": 25,
                "city": "New York"
            }

            print(person["name"])   # John
            print(person["age"])    # 25

            ğŸ”¹ Empty Dictionary

    2.Accessing, adding, updating, and deleting dictionary elements

            âœ… Dictionary Operations: Access, Add, Update, Delete
        ğŸ”¹ Example Dictionary
        student = {
            "name": "Alice",
            "age": 20,
            "grade": "A"
        }

        1ï¸âƒ£ Accessing Elements

        Using the key

        print(student["name"])   # Alice
        print(student["age"])    # 20


        âš ï¸ If the key does not exist â†’ KeyError.

        ğŸ‘‰ Safer way: use .get()

        print(student.get("grade"))     # A
        print(student.get("city"))      # None (no error if key doesnâ€™t exist)

        2ï¸âƒ£ Adding Elements

        Just assign a new keyâ€“value pair

        student["city"] = "New York"
        print(student)
        # {'name': 'Alice', 'age': 20, 'grade': 'A', 'city': 'New York'}

        3ï¸âƒ£ Updating Elements

        Reassign an existing key

        student["age"] = 21   # change age
        print(student)
        # {'name': 'Alice', 'age': 21, 'grade': 'A', 'city': 'New York'}

        student.update({"grade": "A+"})
        print(student)
        # {'name': 'Alice', 'age': 21, 'grade': 'A+', 'city': 'New York'}

        4ï¸âƒ£ Deleting Elements

        student.pop("city")   # removes 'city' key
        print(student)
        # {'name': 'Alice', 'age': 21, 'grade': 'A+'}

        del student["grade"]  # removes 'grade'
        print(student)
        # {'name': 'Alice', 'age': 21}

        student.clear()       # removes all elements
        print(student)

7. Working with Dictionaries

    1.Iterating over a dictionary using loops.

            âœ… Iterating Over a Dictionary

        Use loops to access keys, values, or key-value pairs.

        Letâ€™s take an example dictionary:

        student = {
            "name": "Alice",
            "age": 20,
            "grade": "A"
        }

        ğŸ”¹ 1. Iterating Over Keys (default)
        for key in student:
            print(key)


        Output:

        name
        age
        grade

        ğŸ”¹ 2. Iterating Over Values
        for value in student.values():
            print(value)


        Output:

        Alice
        20
        A

        ğŸ”¹ 3. Iterating Over Key-Value Pairs
        for key, value in student.items():
            print(key, ":", value)


        Output:

        name : Alice
        age : 20
        grade : A

        ğŸ”¹ 4. Using keys() explicitly
        for key in student.keys():
            print(key, "->", student[key])


        Output:

        name -> Alice
        age -> 20
        grade -> A

    2.Merging two lists into a dictionary using loops or zip().

        âœ… Example Lists
        keys = ["name", "age", "grade"]
        values = ["Alice", 20, "A"]

        ğŸ”¹ 1. Using a Loop
        # Create an empty dictionary
        student = {}

        for i in range(len(keys)):
            student[keys[i]] = values[i]

        print(student)


        Output:

        {'name': 'Alice', 'age': 20, 'grade': 'A'}

        ğŸ”¹ 2. Using zip()

        zip() pairs elements from two lists and returns an iterator of tuples.

        student = dict(zip(keys, values))
        print(student)


        Output:

        {'name': 'Alice', 'age': 20, 'grade': 'A'}

        ğŸ”¹ 3. Using Dictionary Comprehension + zip()
        student = {k: v for k, v in zip(keys, values)}
        print(student)


        Output:

        {'name': 'Alice', 'age': 20, 'grade': 'A'}

    3.Counting occurrences of characters in a string using dictionaries.

        âœ… Example: Count Character Occurrences
        text = "hello world"

        # Create an empty dictionary
        char_count = {}

        for char in text:
            if char in char_count:
                char_count[char] += 1   # increment count
            else:
                char_count[char] = 1    # initialize count

        print(char_count)


        Output:

        {'h': 1, 'e': 1, 'l': 3, 'o': 2, ' ': 1, 'w': 1, 'r': 1, 'd': 1}

        ğŸ”¹ Explanation

        Loop through each character in the string.

        Check if character already in dictionary:

        Yes â†’ increment its count

        No â†’ set count to 1

        At the end, the dictionary contains characters as keys and counts as values.

        ğŸ”¹ Alternative: Using get() method
        char_count = {}

        for char in text:
            char_count[char] = char_count.get(char, 0) + 1

        print(char_count)

8. Functions

    1.Defining functions in Python.

         A function is a reusable block of code that performs a specific task.

        Helps organize code, avoid repetition, and make programs modular.

        ğŸ”¹ Syntax

            """
            Optional docstring to describe the function.

        function_name â†’ name of the function (follow naming rules)

        parameters â†’ input values (optional)

        return â†’ value that function sends back (optional)

        ğŸ”¹ Example 1: Function without Parameters
        def greet():
            print("Hello, world!")

        greet()   # calling the function


        Output:

        Hello, world!

        ğŸ”¹ Example 2: Function with Parameters

            print("Hello,", name)

        ğŸ”¹ Example 3: Function with Return Value
        def add(a, b):
            return a + b

        result = add(10, 20)
        print(result)   # 30

        ğŸ”¹ Example 4: Function with Default Parameters
        def greet(name="Guest"):
            print("Hello,", name)

        greet()        # Hello, Guest

    3.Anonymous functions (lambda functions).

        âœ… Anonymous / Lambda Functions

        A lambda function is a small, unnamed function defined using the keyword lambda.

        Used for simple operations that can be written in a single line.

        Can take any number of arguments, but only one expression.

        Returns the value of the expression automatically (no return needed).

        ğŸ”¹ Syntax
        lambda arguments: expression


        arguments â†’ input parameters

        expression â†’ single expression whose value is returned

        ğŸ”¹ Example 1: Add Two Numbers
        add = lambda a, b: a + b
        print(add(10, 20))   # 30

        ğŸ”¹ Example 2: Square a Number
        square = lambda x: x ** 2
        print(square(5))     # 25

        ğŸ”¹ Example 3: Conditional Expression
        max_num = lambda a, b: a if a > b else b
        print(max_num(10, 20))   # 20

        ğŸ”¹ Example 4: Using Lambda with map()
        numbers = [1, 2, 3, 4]
        squared = list(map(lambda x: x**2, numbers))
        print(squared)   # [1, 4, 9, 16]

    1. Introduction to Python modules and importing modules

        âœ… Introduction to Python Modules
        ğŸ”¹ What is a Module?

        A module is a file containing Python code (functions, variables, classes) that you can reuse in other programs.

        Helps in organizing code and avoiding repetition.

        Python has built-in modules, and you can also create your own modules.

        ğŸ”¹ Examples of Built-in Modules

        math â†’ mathematical functions (sqrt, factorial, etc.)

        random â†’ generate random numbers

        datetime â†’ work with dates and times

        os â†’ operating system related operations

        ğŸ”¹ Importing Modules
        1. Import the entire module
        import math

        print(math.sqrt(16))  # 4.0

        2. Import a specific function or variable
        from math import factorial

        print(factorial(5))  # 120

        3. Import with an alias
        import math as m

        print(m.pi)  # 3.141592653589793

        4. Import all functions (not recommended)
        from math import *

        print(sqrt(25))  # 5.0


        âš ï¸ Note: Using * may overwrite existing functions or variables.

        ğŸ”¹ Creating Your Own Module

        Create a Python file, e.g., my_module.py:

        def greet(name):
            print("Hello,", name)


        Import it in another file:

        import my_module

        my_module.greet("Alice")  # Hello, Alice

            âœ… Creating Custom Modules in Python

            A module is basically a Python file (.py) containing functions, variables, or classes that you can reuse in other programs.
